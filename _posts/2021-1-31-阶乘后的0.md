###       阶乘后的0

遍历从1到n，找出每个数因子为5的个数。

因为只要有一个5，就有一个2，2的因字数大于等于5的因子数。

<pre name="code" class="c++"> 
int trailingZeroes(int n) {
        int count = 0;
        for(int i = 1; i <= n; i++){
            int k = i;
            while(k>=5 && k%5==0){ //找出i的因子5的个数
                 k/=5;
                 count++;
            }
        }
        return count;
    }
</pre>
