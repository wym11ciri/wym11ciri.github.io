题目描述：

给定一个整数数组 nums ，找到一个具有最大和的连续子数组（子数组最少包含一个元素），返回其最大和。

首先设置sum = nums[0]

ans = 0        ans用来记录最大的sum

如果前面的和是小于零的，那么就舍弃不要，把sum置为nums[i]。

不然就sum+=nums[i]

```
int maxSubArray(vector<int>& nums) {
        if(nums.size()==1) return nums[0];
        int ans = nums[0];
        int sum = 0;
        for(int i = 0; i < nums.size(); i++){
            if(sum > 0) sum += nums[i];
            else sum = nums[i];
            ans = max(ans, sum);
        }
        return ans;
    }
```
